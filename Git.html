Subject : Git & GitHub এর মধ্যে গিট্টু লাগানো।

***প্রথমবার GitHub ডাটা পুশ করার নিয়ম।প্রথমবার ডাটা পুশ করার জন্য ৬ টি ধাপে ডাটা পুশ করতে হয়।

  1. প্রথমে ডাটা Initialization করেতে হবে। উদাহরণঃ >Git init

  2. দ্বিতীয় ধাপে গিট এ ডাটা অ্যাড করে নিতে হবে। উদাহরণঃ >Git add file name(যে ফাইল অ্যাড করব তার নাম অথবা . দিয়ে সবগুলো ফাইল অ্যাড করে নিতে পারি) >Git add .

  3. তৃতীয় ধাপে Git Commit করে নিতে হবে। উদাহরণঃ >Git commit -m"Commit Name"

  4. চার নাম্বার ধাপে Branch পরিবর্তন করে Main Branch নিতে হবে। উদাহরণঃ >Git branch -M Main

  5. পাচ নাম্বার ধাপে Remote এ ডাটা অ্যাড করে নিতে হবে। অর্থাৎ যেখানে ডাটা পুশ করতে চাচ্ছেন ওই গিট হাবের URL. উদাহরণঃ >Git remote add URL

  6. সবশেষ এখন শুধু ডাটা পুশ করলেই গিট হাব এ ডাটা পুশ হয়ে যাবে। উদাহরণঃ >git push -u origin main

  1. প্রথম ধাপে গিট এ ডাটা অ্যাড করে নিতে হবে। উদাহরণঃ >Git add file name(যে ফাইল অ্যাড করব তার নাম অথবা . দিয়ে সবগুলো ফাইল অ্যাড করে নিতে পারি) >Git add .

  2. দ্বিতীয় ধাপে Git Commit করে নিতে হবে। উদাহরণঃ >Git commit -m"Commit Name"

  3. সবশেষ এখন শুধু ডাটা পুশ করলেই গিট হাব এ ডাটা পুশ হয়ে যাবে। উদাহরণঃ >git push
